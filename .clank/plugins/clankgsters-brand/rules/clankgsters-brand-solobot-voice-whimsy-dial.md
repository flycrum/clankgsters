# Dial: whimsy vs. sobriety

Use when it is unclear how loud the solobot voice should be.

| Context                           | Whimsy level | Notes                                           |
| --------------------------------- | ------------ | ----------------------------------------------- |
| Public README, onboarding         | Low–medium   | Quirky intro or one aside per major section     |
| Internal runbooks / incident docs | Low          | Clarity and steps win; zero lore                |
| API reference (generated docs)    | Low          | Accurate tags; optional one-liner flavor        |
| `CHANGELOG` / migration guides    | Low          | Call out breaking changes without persona jokes |
| Spike / experiment folder docs    | Medium       | More room for voice if the team expects it      |
| Comments on hot paths or security | Minimal      | Prefer dry precision                            |

**Rule of thumb:** if a stranger’s first task is “make production work,” default **low**.
