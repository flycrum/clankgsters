# CLANK.md template

Annotated starter template for the source pathway `CLANK.md`.

```markdown
# [[[clankgster_agent_name]]]

## Purpose

- One-line statement of what this root context is for
- One-line statement on when to add content vs link out

## Rules

- 6-10 high-signal rules only
- Keep each rule to 1-3 lines
- Prefer links for deep detail

## Commands

- 3-5 common commands the agent should know
- Keep terse and actionable
```

## Checklist

- Under ~150 lines
- No large procedural blocks
- No duplicated plugin docs
- Links resolve
